The Site Locator Map plugin is a location management system that allows your users to locate places, stores, spots and other areas through search filters.

Installation

There are several methods to install a WordPress plugin. Choose a method below.

New Install via FTP

  1. Unzip the package site-locator-map-x.x.zip on you computer.
  2. Upload the unzipped folder the /wp-content/plugins/ directory.
  3. Activate the plugin through the Plugins menu in WordPress.
  4. Go to Site Locator Map > Options to set your options.

New Install via the WordPress uploader

  1. Click Plugins > Add New inside of your WordPress install.
  2. Click Upload and select the package site-locator-map-x.x.zip
  3. Activate the plugin through the Plugins menu in WordPress.
  4. Goto Site Locator Map > Options to set your options.

Updating via the FTP

  1. If you've made any custom changes to the core you'll need to merge those changes into the new package.
  2. Unzip the package site-locator-map-x.x.zip on you computer.
  3. Replace the current folder in your plugins folder with the new unzipped folder from your computer.

Updating via the WordPress uploader

  1. Navigate to the Plugins page in WordPress and delete the current plugin.
  2. Now follow the New Install via the WordPress uploader instructions.

Features

Map

Template Overrides

Location Information

Adding a location

Go to Site Locator Map > Add New Location

Adding a title will display the location title. Below you can add the location map information under the Location Map metabox.

Input the address of your location by typing in the Address: input box and click Get Data button to get the longitude and latitude of the location.

If you only have the longitude and latitude type those in thier input boxes and click the Get Data button, it will automaticaly show the address of the location in the Address: input box.

You can also drag the pin icon to get the actual point of the location.

 

Adding a location category

Go to Site Locator Map > Location Categories

Type in the name, slug, description and select an image for the marker icon (check Marker Icon instruction section).

Then click Add New Category button. This will add a category for locations.

 

Assigning a category to a location

Go to Site Locator Map > All Locations choose a location by clicking on edit.

At the right hand site there is a box where you tick location category tick a category for that location and then click publish.

 

Adding fields to the location

Adding fields to the location will display more information to the location. This will be shown as meta box in the single admin location page. The fields will be displayed in the map list and in the map info window.

Go to Site Locator Map > Options click on the Fields tab. You will see that a Description field is added by default.

To add more fields click on the Add button.  

Type in the field name for the field and select a field type if you want the field to be shown as text area choose multiple or if you want an input box choose single. 

You can also hide the field in the map info window by choosing No in the Show on Map Info Window section.

The fields are draggable to order the display of the fields in the front end.

 

 

Marker Icon

Category Marker Icon

You can change the default marker icon for each category location. 

Go to Silte Locator Map > Location Categories

Upon adding a new category click on the Add Image button at the bottom left side under image section. This will open up the wordpress media library. You can upload a new image or choose an image from the existing media library.

When editing the an existing icon just choose a category in Locator Map > Location Categories click edit. When you are on the category page click on Remove Image and click on Add Image button. This will open up the wordpress media library. You can upload a new image or choose an image from the existing media library.

Shortcodes

Paste this shortcode to your post or page:

[site_locator_map]

 

You can add shortcode parameters to change the map's look and behaviour:

Example:

[site_locator_map type=SATELLITE align=right width=600]

The example above aligns the map on the right and will display the map in satellite view with a width of 600px.

 

The shortcode parameters will override configuration in Site Locator Map > Options

Shortcode parameters:

zoom - input a number from 1 - 20 | e.g. 12

width - the number that will be inserted will be converted as px. | e.g. 400

height - same as width. | e.g. 500

align -  aligns the map to the left, center or right | values: "left", "center", "right"

type - changes the map view | values: "SATELLITE", "ROADMAP", "HYBRID"

mousewheel - disables map mouse wheel support. Set false to disable the support. | values: "true", "false"

pancontrol - disables map pan control support. Set false to disable the support. | values: "true", "false"

scalecontrol - hides the map scale control support. Set false to disable the support. | values: "true", "false"

typecontrol - hides the map scale control support. Set false to disable the support. | values: "true", "false"

showlist - set false to hide map results list display | values: "true", "false"

showsearch - set false to hide map search display | values: "true", "false"

showmap - set false to hide google map display | values: "true", "false"

perpage - input a number to show number of resutls per page: e.g. 25

message - This will hide result location count when set to false | values: "true", "false"
 
ext_url - You can redirect the search form to another page by adding slug to that parameter.

 

You can also change the map locaiton results by adding shortcode parameters.

Category parameter:

[site_locator_map category=6]

This shortcode will display map locations with a location category id or tag id of 6

 

Single location parameter:

You can display single map result by passing a location id.

[site_locator_map location_id=27]

 

Search Form Only

[site_locator_map_form ext_url="redirect"]

Displays search form only with ext_ul for the form redirection.

Template Overriding

Template Overrides

  • Template files of Site locator map contain the markup and template structure for the front-end (and HTMLs) of your site that can be easily change.
  • If you want to change the layout of the map info window Go to site-locator-map plugin folder open templates copy location-infowindow.php to your theme root folder. In the file you can change the layout and add more contents.

Here are the files that can be overridden (same process follows):

location-content.php - overrides the location content result list.

location -search_form.php - overrides the map search form. You can re order the search form template tags to style it.

archive-location.php - overrides the default location archive location template which has the default url www.yourdomain.com/location

single-location.php - overrides the default single location template. 

taxonomy-location_category.php - overrides the default location taxonomy template which has the default url www.yourdomain.com/location_category/{term}

 

Template Tags

Template tags are used within your site locations templates to display information dynamically or otherwise customize the layout of the page or site.

slm_result_message() - Displays the number of location results.

slm_map() - Displays the location google map.

slm_search_form() - Displays the location search form.

slm_paginate() - Displays the location pagination results.

slm_display_fields() - Displays all location fields that has been set at the Site Locator Map > Options > Fields

get_slm_field($field_name) - The function accepts one parameter the field name you have created. Passing "address", "longitude" or "latitude" will ge the location map information.

the_slm_field($field_name) - Automaticaly echoes get_slm_field.

slm_get_location_attributes() - Displays all location attributes.

Advance

Changing the default archive url location which www.yourdomain.com/location

Use this wordpress filter:

function slm_slug() {
 return "area";
}
add_filter("slm_slug", "slm_slug");
 
After you add that snippet update your permalinks by going to your wordpress admin Settings > Permalinks save changes.
 
The example will change the default url to www.yourdomain.com/area
 
 

Applying Site Locator Map to an existing post type

Use this wordpress filter:
 
function slm_post_type() {
return "area";
}
add_filter("slm_post_type", "slm_post_type");

The example will add Site Locator Map feature to area post type.
