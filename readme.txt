A location management system that allows your users to locate places, stores, spots and other areas through search filters.



== Description ==

<h3>Features</h3>
<h4>Map</h4>
<ul>
	<li>
		Multiple markers</li>
	<li>
		Multiple maps in one page</li>
	<li>
		Changeable map icon marker.</li>
	<li>
		Marker Clusterer. Solution for too many markers.</li>
	<li>		Marker info window.</li>
</ul>

<h4>Map Search</h4>
<ul>
	<li>
		Auto complete search address.</li>
	<li>
		Search radius either kilometers or miles.</li>
	<li>
		Search by location category.</li>
	<li>
		Search Form Widget.</li>
</ul>

<h4>Location Information</h4>
<ul>
	<li>
		Easily add location lattitude and longitude through google map meta box.</li>
	<li>
		Add location fields that will be shown as metabox.</li>
	<li>
		Can be shown in info window.</li>
	<li>
		Set location results pagination.</li>
</ul>

<h4>Map shortcode</h4>
<ul>
	<li>
		Add map shortcodes in posts and pages.</li>
	<li>
		Set map category and attributes through shortcode parameters. Overrides back end options .</li>
	<li>
		Set map options through shortcode parameters. Override back end options.</li>
</ul>

<h4>Template Overrides</h4>
<ul>
	<li>
		Template files of Site locator map contain the markup and template structure for the front-end (and HTMLs) of your site that can be easily change.</li>
</ul>

Files include Api calls to Google Map API Web Services.

For more features of the plugin purchase the paid version here: http://codecanyon.net/item/site-locator-map/7354406
== Installation ==
1. Upload 'site-locator-map-free' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add locations in Site Locator Map Free -> Add locations
4. Use shortcode [site_locator_map]
For more option and setup instructions of the plugin please open up the included documentation folder.
